Microtubule length distribution analysis tool

*This analysis tool is part of the MT analysis program, which is work in progress.

Instruction:

Upload file:

The MT analysis tools working with Amira ZIB MT spatial graph file (.am).

MT spatial graph should contain Amira length analysis (using Spindle Analysis -> Length tool), and an exported .xml file should be converted to .xlsx (excel document file)class=""

To upload files for the analysis, move to the "Upload" are on your left. This tool supports up to 4 files.

When the file will be properly uploaded to the server, on the right side you will be able to validate the excel table.

Analysis:

MT Length Distribution tool is able to recognize kinetochore microtubules (KMTs) from Non-KMTs. For this algorithm is searching for MT with labels starting with I) "Pole", or II) KMTs.

This tool is performing two operations on your data I) Length Histogram distribution, or II) Percanteg of MT to length distribution

Export data:

To export data, simply move to "Export" part of the tool, select which data to export and download .csv file.

The downloaded file will contain data for selected analysis for each uploaded file, as well as average data if more then two files were uploaded
